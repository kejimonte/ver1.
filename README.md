## 本サービスの概要（700文字以内）

本サービスは、「自分のための感謝ログ」をコンセプトにした、感謝の気持ちを記録・振り返るためのWebアプリケーションです。
<br>日々忙しく働く20〜30代の社会人を主な対象とし、仕事や人間関係に追われる中で「気持ちを言語化する場所がない」「前向きな感情に目を向ける余裕がない」といった課題を解決することを目的としています。

ユーザーは日常の中で感じた小さな感謝や出来事を「感謝投稿」として記録できます。
<br>投稿にはテキストだけでなく、画像やタグを付けることができ、自分の気持ちを整理しやすい設計になっています。
<br>過去の投稿を一覧や詳細画面で振り返ることで、自分がどんな瞬間に感謝してきたのかを可視化できます。

また、投稿は公開・非公開を選択でき、公開された投稿は未ログインユーザーでも閲覧可能です。
<br>他ユーザーの投稿を読むことで、新たな気づきや共感を得ることもできます。

本サービスは、他人と競うSNSではなく、「自分自身と向き合う時間」を大切にし、感謝を習慣化することで心の余裕を取り戻す体験を提供します。


## READMEに記載した機能
■MVPリリース時
- [☑]ユーザー登録 / ログイン
- [☑]ありがとう日記の投稿（テキスト）
- [☑]一覧表示 / 詳細表示
- [☑]日付ごとの投稿管理

■本リリースまでに実装したい機能
- [-]タグ付け（人・出来事・感情など）
- [-]カレンダー表示
- [-]ランダム振り返り機能
- [-]投稿の非公開 / 公開設定
- [-]シンプルな分析（投稿数の推移など）

※ 本サービスでは「記事」を「感謝投稿」として扱います。


## 未ログインでも閲覧または利用できるページ

以下の項目は、未ログイン状態でも閲覧可能な画面遷移としています。

- 記事閲覧機能（未ログインでも閲覧可能）  
- コメント閲覧機能（未ログインでも閲覧可能）  

※ 記事投稿、コメント投稿、編集、削除などの操作はログイン必須です。


## メールアドレス・パスワード変更確認項目

セキュリティを考慮し、メールアドレスおよびパスワードの変更は直接行わず、以下のフローで変更できる設計としています。

### メールアドレス変更
変更申請 → 登録メールアドレス宛に確認用URL送信 → 専用ページで変更

### パスワード変更
変更申請 → 登録メールアドレス宛に確認用URL送信 → 専用ページで変更
